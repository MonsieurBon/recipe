# 12. Glossary

| Term | Definition |
|---|---|
| Recipe | A cooking recipe with a title, ingredients, steps, and metadata like season |
| Favorite | A user's bookmarked recipe, given priority in meal suggestions |
| Meal Plan | A weekly schedule assigning recipes to specific meal slots |
| Meal Slot | A specific meal occasion (e.g. Monday dinner) within a meal plan |
| Season | A time-of-year classification for recipes (e.g. spring, summer, autumn, winter, all-year) |
| Suggestion | An automatically generated meal assignment based on season, history, and preferences |
