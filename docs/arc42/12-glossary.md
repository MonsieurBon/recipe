# 12. Glossary

| Term | Definition |
|---|---|
| Recipe | A cooking recipe with a title, ingredients, steps, and metadata |
| Ingredient | A reusable cooking ingredient (e.g. asparagus, salt) with optional month-based seasonality |
| Recipe Ingredient | The association of an ingredient to a recipe with a specific quantity (amount + unit) |
| Step | A single instruction within a recipe, which can reference the recipe ingredients it uses |
| Favorite | A user's bookmarked recipe, given priority in meal suggestions |
| Meal Plan | A weekly schedule assigning recipes to specific meal slots |
| Meal Slot | A specific meal occasion (e.g. Monday dinner) within a meal plan |
| Season Months | The months during which an ingredient is in season; if unset, the ingredient is considered in season year-round |
| Suggestion | An automatically generated meal assignment based on ingredient seasonality, history, and preferences |
