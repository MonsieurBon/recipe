# 10. Quality Requirements

## 10.1 Quality Scenarios

| Scenario | Quality | Expected Response |
|---|---|---|
| User opens a recipe while cooking | Usability | Recipe is displayed within 1 second, readable on a phone screen |
| User requests meal suggestions | Performance | Suggestions are generated within 2 seconds |
| User adds a recipe with incomplete data | Reliability | Validation errors are shown; no data is lost |
| Developer adds a new feature | Maintainability | Clear package structure; existing tests pass; new feature can be added with localized changes |
